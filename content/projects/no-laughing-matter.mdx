overview: "No Laughing Matter is a native macOS application built in Swift that fetches the official plenary protocols of the German Bundestag. It then parses, classifies, and visualizes humor events. The program covers legislative periods 18 through 21 (2013–present) and has processed over 15,900 individual humor events across thousands of plenary sessions."
approach: "The processing pipeline consists of four stages:
1. Protocol Fetcher. The protocols are fetched via the official DIP-API of the German Bundestag.
2. Event Parser. A RegEx-Pattern is applied to detect humor markers such as „Heiterkeit“ (amusement) and „Lachen“ (laughter). The parser also extracts speaker metadata – party, role, date and topic – and enriches each event with age and gender data from the official MdB database.
3. LLM Classifier. Each event, including surrounding context is sent to an LLM via the Claude Batch API for function classification according to Avner Ziv’s humor function model.
4. Visualization. The program includes a dedicated visualization tab for reviewing and exporting results. This uses Swift Charts."
